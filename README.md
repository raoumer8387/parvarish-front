# Parvarish AI - Frontend

A React + TypeScript + Vite application for Parvarish AI parenting platform.

## Features

- Built with React 18 and TypeScript
- Vite for fast development and building
- Tailwind CSS v4 for styling
- Radix UI components for accessible UI
- Login page and dashboard views
- Chatbot interface
- Activities and games
- Progress tracking

## Getting Started

1. **Install dependencies**

   ```powershell
   npm install
   ```

2. **Start development server**

   ```powershell
   npm run dev
   ```

3. **Open the app**
   
   The app will open automatically at http://localhost:3000

## Project Structure

- `src/`
  - `components/` - React components
    - `ui/` - Reusable UI components (Radix UI based)
    - `LoginPage.tsx` - Login page
    - `ParentDashboard.tsx` - Parent dashboard
    - `ChatbotInterface.tsx` - AI chatbot
    - `ActivitiesPage.tsx` - Activities listing
    - `GameScreen.tsx` - Game interface
    - `ProgressDashboard.tsx` - Progress tracking
    - `SettingsPage.tsx` - Settings
  - `styles/` - Global styles
  - `App.tsx` - Main app component
  - `main.tsx` - Entry point

## Technologies

- React 18.3
- TypeScript 5.3
- Vite 5.4
- Tailwind CSS v4
- Radix UI
- Lucide React (icons)
- Recharts (charts/graphs)
