# Parvarish Front (minimal React + Vite)

This is a minimal React + Vite scaffold created by an automated assistant.

Getting started

1. Install dependencies

   npm install

2. Start dev server

   npm run dev

3. Open the URL printed by Vite (default http://localhost:5173)

Files

- `index.html` - app entry
- `src/main.jsx` - React entry
- `src/App.jsx` - main app component
- `src/index.css` - basic styles
